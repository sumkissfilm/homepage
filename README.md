# MASASTAKE 改裝車品牌網站

## 專案簡介

這是一個為 MASASTAKE 改裝車品牌設計的現代化響應式網站首頁。網站採用最新的網頁技術，提供優質的用戶體驗和視覺效果。

## 功能特色

### 🎨 設計特色
- **現代化設計**：採用漸層色彩和動畫效果
- **響應式設計**：完美適配桌面、平板和手機設備
- **品牌色彩**：使用橙色系主色調，符合改裝車品牌形象
- **動畫效果**：包含滾動動畫、懸停效果和打字機效果

### 📱 響應式功能
- **4K 標準設計**：以 2560px 為標準大小，hero高度1347px，完美適配 4K 顯示器
- **長寬比優化**：維持 1.9:1 的長寬比，確保視覺一致性
- **QHD 高效能設計**：以 1440px 為高效能顯示器優化，適合電競和專業工作
- **1080p 通用設計**：以 1080px 為通用大小，覆蓋主流桌面設備
- **多斷點設計**：支援從 4K 到手機的全尺寸適配
- **移動端漢堡選單**：觸控友好的導航體驗
- **自適應網格佈局**：CSS Grid 和 Flexbox 完美配合
- **觸控手勢支援**：滑動手勢和觸控優化
- **效能優化**：懶加載和動畫效能優化
- **無障礙支援**：鍵盤導航和螢幕閱讀器支援

### 🚀 互動功能
- 平滑滾動導航
- 表單驗證和提交
- 滾動進度指示器
- 數字計數器動畫
- 視差滾動效果

## Hero 背景圖片

網站使用自訂背景圖片，位於 `foto/hero-bg.jpg`：

### 背景圖片設定
- **圖片路徑**: `foto/hero-bg.jpg`
- **覆蓋層**: 深色漸層覆蓋確保文字可讀性
- **響應式**: 自動適配所有螢幕尺寸
- **固定位置**: 背景圖片不隨滾動移動

### 更換背景圖片
1. 將您的背景圖片命名為 `hero-bg.jpg`
2. 放置在 `foto/` 資料夾中
3. 建議圖片尺寸：至少 1920x1080 像素
4. 支援格式：JPG、PNG、WebP

### 自訂背景圖片
如需使用不同的圖片名稱，請修改 `css/styles.css` 中的路徑：

```css
.hero {
    background-image: 
        linear-gradient(135deg, rgba(26, 26, 26, 0.8) 0%, rgba(45, 45, 45, 0.8) 100%),
        url('../foto/您的圖片名稱.jpg');
}
```
